# Project Summary
The DIDECO Social Aid Management System is designed to streamline the delivery of social aid to beneficiaries. It features modules for managing aid delivery, generating reports, and maintaining inventory, all integrated with a user-friendly interface. The system aims to enhance efficiency and transparency in social aid distribution.

# Project Module Description
- **Aid Delivery Module**: Allows users to register the delivery of social aid, search for beneficiaries, and generate printable receipts.
- **Reports Module**: Provides a dashboard with insights into inventory, critical stock levels, and aid delivery history, including data export capabilities.

# File Description Inventory
- **src/App.tsx**: Main application component that includes routing.
- **src/pages/AidDelivery.tsx**: Interface for delivering social aid.
- **src/pages/Reports.tsx**: Dashboard for viewing reports and inventory.
- **src/components/aid/ReceiptModal.tsx**: Modal for displaying delivery receipts.
- **src/store/useAppStore.ts**: Centralized state management for the application.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Fast build tool for modern web applications.
- **Zustand**: State management library.

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the project:
   ```
   pnpm run start
   ```
